CSS Selectors

Universal Selector

universal.html 
<!DOCTYPE html>
<html>
    <head>
        <title>Universal Selector</title>
        <style>
            *{
                color: teal;
            }
        </style>
    </head>
    <body>
        <h1>Universal selector</h1>
        <p>Universal selector selects all elements in HTML</p>
    </body>
</html>
Output

ID Selector

id-Selector.html 
<!DOCTYPE html>
<html>
    <head>
        <title>ID Selector</title>
        <style>
            #head{
                border: 2px solid red;
                background-color: aqua;
            }
            #para {
                color: red;
            }
        </style>
    </head>
    <body>
        <h1 id="head">ID Selector</h1>
        <p id="para">ID selector selects elements with specified ID</p>
        <p>ID should be unique</p>
    </body>
</html>
Output

    <body>
        <form>
            ID: <input type="text" placeholder="Enter your id">

</html>
